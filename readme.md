## 📂 Projects

### 🎮 Games

- **Tic Tac Toe**  
  A two-player X-O game where players take turns.

- **Memory Game**  
  An interactive game where the player finds matching flower pairs  
  in the shortest time and with the fewest moves.

- **Sudoku**  
  A classic Sudoku game with multiple difficulty levels.

- **Pac-Man**  
  A classic Pac-Man game controlled with arrow keys.  
  The goal is to eat all the dots without touching the ghosts.

- **Taki Game**  
  A two-player interactive card game based on the original Taki rules.

### 🧮 Application
